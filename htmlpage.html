What is HTML?

HTML is a Hypertext Markup Language,a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.

What is CSS?

CSS means Cascading Style Sheets (CSS) and is a style sheet language used for describing the look and formatting of a document written in a markup language, like the skin of the website.

How do Tags Work?

In information systems, a tag is a non-hierarchical keyword or term assigned to a piece of information (such as an Internet bookmark, digital image, or computer file). This kind of metadata helps describe an item and allows it to be found again by browsing or searching.

What are attributes and how do they work?

Attributes are pieces of information that determine the properties of a field or tag in a database or a string of characters in a display.

Some tags used daily are...

  • A href tag which he href attribute specifies the URL of the page the link goes to.If the href attribute is not present, the a tag is not a hyperlink.
  • A div tag which defines a division or a section in an HTML document.The div tag is used to group block-elements to format them with CSS.
  • A head tag is important The element is a container for all the head elements.The element must include a title for the document, and can include scripts, styles, meta information, and more.The following elements can go inside the head element: (this element is required in the head section)style,base,link.meta,script,noscript.
  • A p tag because the p tag defines a paragraph.Browsers automatically add some space (margin) before and after each p element. The margins can be modified with CSS (with the margin properties).
